\# Context7 MCP



\## 1. Objetivo



O \*\*Context7 MCP\*\* foi configurado para fornecer ao Codex acesso a documentações técnicas atualizadas de bibliotecas, frameworks e ferramentas utilizadas no projeto \*\*quarta-etapa-chamados\*\*.



No contexto deste projeto, o Context7 deve ser usado principalmente para consultar documentação atual de:



\- Next.js App Router

\- Supabase Auth

\- Supabase SSR

\- Vercel

\- Bibliotecas React/TypeScript relevantes

\- Ferramentas de apoio ao desenvolvimento



O objetivo é reduzir decisões baseadas em documentação desatualizada e apoiar implementações com base em fontes atuais.
