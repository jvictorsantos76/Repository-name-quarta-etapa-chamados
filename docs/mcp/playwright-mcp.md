\# MCP Playwright / Browser



\## Objetivo



O MCP Playwright fornece automação de navegador para validação funcional e visual do projeto quarta-etapa-chamados.



Ele será usado para testar fluxos reais da aplicação, como login, abertura de chamados, preenchimento de formulários, validações de campos, navegação e mensagens exibidas ao usuário.



\## Aplicação no projeto



Uso previsto:



\- Validar tela de login.

\- Testar abertura de chamados.

\- Verificar campos obrigatórios.

\- Conferir navegação entre páginas.

\- Capturar evidências visuais quando necessário.

\- Apoiar testes de regressão após alterações relevantes.

\- Avaliar comportamento básico em diferentes resoluções.
